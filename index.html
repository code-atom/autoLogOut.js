autoSignOut.js

Use localstorage for update the last user activity with unix-timestamp

Browser Issue

autoSignOut.js implements the event listener on following events:-

  1. Document ready
  2. Document unload
  3. Mouse movement
  4. Scroll event
  5. Window resize
  6. Storage change
All events are throttle with setTimeOut.

For implementation

 
       var signOut = autoSignOut.init({ timeout: 1, sessionKey: 'Key', debug: true});
       signOut.onSignOut = function () {
           alert("Called Timeout");
           console.log("Timeout Called");
       }

autoSignOut.init() function take intial configuration object, each key description mentioned below:-

        timeout: number of minutes after that session will auto logout
        sessionKey: name of the localstorage key
        debug: console.log on event called and timeout registration.

autoSignOut.js have three methods:-

  1. init : initial configuration function
  2. signOut: mannual call the onSignOut callback
  3. onSignOut: this callback callback on logout
